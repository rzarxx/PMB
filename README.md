# Portal Pendaftaran Mahasiswa Baru - IPB Cirebon

Website ini merupakan **Tugas Ujian Akhir Semester (UAS)** untuk mata kuliah **Design Web**. Proyek ini dikerjakan oleh **Kelompok 4**.

## Deskripsi Proyek
Proyek ini adalah desain ulang (redesign) antarmuka pengguna (UI/UX) untuk portal Pendaftaran Mahasiswa Baru (PMB) khusus untuk **IPB Cirebon**. Kami merancang website ini untuk menjadi lebih modern, profesional, intuitif, dan responsif (dapat digunakan dengan mulus di berbagai perangkat dari desktop hingga mobile).

Website ini dibangun menggunakan:
- **HTML5** (Semantik & Struktur Layout)
- **CSS3** (Custom Properties / CSS Variables, Flexbox, Grid, tanpa Framework)
- **JavaScript Vanilla** (Untuk interaktivitas seperti Hamburger Menu dan Collapse Sidebar)
- **Google Fonts** (Work Sans & Source Serif 4)
- **Material Symbols Outlined** (Untuk ikonografi UI)

## Daftar Halaman (Pages)
Proyek ini mencakup 11 halaman utama, antara lain:
1. `index.html` - Halaman Beranda (Landing Page)
2. `login.html` - Halaman Autentikasi / Masuk
3. `register.html` - Halaman Pendaftaran Akun Baru
4. `jalur-mandiri.html` - Detail Jalur Masuk Mandiri
5. `jalur-prestasi.html` - Detail Jalur Masuk Prestasi
6. `jalur-kip.html` - Detail Jalur Masuk KIP-Kuliah
7. `dashboard.html` - Dashboard Utama Peserta / Overview
8. `dashboard-biodata.html` - Pengisian Biodata Diri & Data Akademik
9. `dashboard-prodi.html` - Pemilihan Program Studi
10. `dashboard-ortu.html` - Pengisian Data Orang Tua / Wali
11. `dashboard-upload.html` - Unggah Dokumen Persyaratan

## Fitur Utama UI/UX
- **Desain Kelas Premium**: Identitas visual khas universitas (IPB Cirebon) dipadukan dengan desain web modern layaknya dashboard aplikasi *enterprise*.
- **Responsive Web Design**: Sidebar pada dashboard bisa menciut (minimize) di Desktop untuk memberikan ruang konten penuh, dan berubah menjadi *overlay pop-up* saat dibuka melalui HP.
- **Micro-interactions**: Efek transisi halus (hover effects) pada tombol, *input fields*, kartu prodi (radio cards), hingga notifikasi pendaftaran.
- **Zero-friction Layout**: Form-form kompleks dibuat rapi melalui pembagian *Section* (*Card*) agar pendaftar nyaman membaca tanpa merasa kelelahan kognitif.
